Fundamentos de las Pruebas de Hipótesis

¿Qué es una Prueba de Hipótesis?

Una prueba de hipótesis es un procedimiento estadístico que permite tomar decisiones sobre parámetros poblacionales basándose en información muestral.

Elementos fundamentales:

  • Hipótesis Nula (\(H_0\)): Afirmación sobre el parámetro poblacional que se asume verdadera
  • Hipótesis Alternativa (\(H_1\) o \(H_a\)): Afirmación que se acepta si hay evidencia suficiente contra \(H_0\)
  • Nivel de significancia (\(\alpha\)): Probabilidad de rechazar \(H_0\) cuando es verdadera (típicamente 0.05 o 0.01)
  • Estadístico de prueba: Valor calculado a partir de los datos muestrales
  • Región crítica: Valores del estadístico que llevan a rechazar \(H_0\)
  • p-valor: Probabilidad de obtener un resultado al menos tan extremo como el observado, asumiendo que \(H_0\) es verdadera

Tipos de Hipótesis Alternativas

Tipo de Prueba Hipótesis Nula Hipótesis Alternativa Región Crítica
Bilateral (dos colas) \(H_0: \theta = \theta_0\) \(H_1: \theta \neq \theta_0\) Ambos extremos de la distribución
Unilateral derecha \(H_0: \theta \leq \theta_0\) \(H_1: \theta > \theta_0\) Extremo derecho (cola superior)
Unilateral izquierda \(H_0: \theta \geq \theta_0\) \(H_1: \theta < \theta_0\) Extremo izquierdo (cola inferior)

Procedimiento de Prueba de Hipótesis

Paso 1: Plantear las Hipótesis

Formular claramente \(H_0\) y \(H_1\) en términos del parámetro poblacional.

Paso 2: Elegir el Nivel de Significancia

Establecer el valor de \(\alpha\) (comúnmente 0.05, 0.01 o 0.10).

Paso 3: Seleccionar el Estadístico de Prueba

Elegir el estadístico apropiado según el parámetro y las condiciones:

  • Prueba Z: Para la media con \(\sigma\) conocida o \(n\) grande
  • Prueba t: Para la media con \(\sigma\) desconocida y \(n\) pequeña
  • Prueba Z para proporción: Para proporciones con \(n\) grande

Paso 4: Calcular el Valor del Estadístico

Usar los datos muestrales para calcular el valor del estadístico de prueba.

Paso 5: Tomar la Decisión

Comparar el estadístico calculado con el valor crítico o comparar el p-valor con \(\alpha\):

  • Si \(p < \alpha\): Rechazar \(H_0\)
  • Si \(p \geq \alpha\): No rechazar \(H_0\)

Paso 6: Conclusión

Interpretar el resultado en el contexto del problema.

Tipos de Errores

\(H_0\) es Verdadera \(H_0\) es Falsa
No rechazar \(H_0\) Decisión correcta (1 - \(\alpha\)) Error Tipo II (\(\beta\))
Rechazar \(H_0\) Error Tipo I (\(\alpha\)) Decisión correcta (Potencia = 1 - \(\beta\))

Error Tipo I (\(\alpha\)): Rechazar \(H_0\) cuando es verdadera (falso positivo)

Error Tipo II (\(\beta\)): No rechazar \(H_0\) cuando es falsa (falso negativo)

Potencia de la prueba (1 - \(\beta\)): Probabilidad de rechazar correctamente \(H_0\) cuando es falsa

Fórmulas Principales

1. Prueba Z para la Media (σ conocida)

$$Z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}$$

donde \(\bar{x}\) es la media muestral, \(\mu_0\) es el valor bajo \(H_0\), \(\sigma\) es la desviación estándar poblacional, y \(n\) es el tamaño de muestra.

2. Prueba t para la Media (σ desconocida)

$$t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}$$

donde \(s\) es la desviación estándar muestral. Se distribuye como t-Student con \(n-1\) grados de libertad.

3. Prueba Z para la Proporción

$$Z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}$$

donde \(\hat{p} = \frac{x}{n}\) es la proporción muestral, \(p_0\) es la proporción bajo \(H_0\), y \(x\) es el número de éxitos.

📏 Prueba de Hipótesis para la Varianza (σ²)

Distribución Chi-cuadrado (χ²): Se utiliza para probar hipótesis sobre la varianza o desviación estándar de una población normal.

Estadístico de Prueba:

$$\chi^2 = \frac{(n-1)s^2}{\sigma_0^2}$$

donde:

  • \(n\) = tamaño de muestra
  • \(s^2\) = varianza muestral
  • \(\sigma_0^2\) = varianza bajo H₀
  • Grados de libertad: \(df = n - 1\)

Hipótesis:

Bilateral: \(H_0: \sigma^2 = \sigma_0^2\) vs \(H_1: \sigma^2 \neq \sigma_0^2\)

Unilateral derecha: \(H_0: \sigma^2 \leq \sigma_0^2\) vs \(H_1: \sigma^2 > \sigma_0^2\)

Unilateral izquierda: \(H_0: \sigma^2 \geq \sigma_0^2\) vs \(H_1: \sigma^2 < \sigma_0^2\)

⚠️ Nota importante: Esta prueba es sensible a la violación del supuesto de normalidad. Asegúrese de que la población sea aproximadamente normal antes de aplicar esta prueba.

Pruebas de Hipótesis para Dos Poblaciones

📊 Comparación de Dos Poblaciones

Las pruebas de hipótesis para dos poblaciones permiten comparar dos grupos independientes o relacionados para determinar si existe una diferencia estadísticamente significativa entre ellos.

Tipos de Pruebas para Dos Poblaciones

Tipo de Prueba Parámetro Condiciones Estadístico
Dos medias (muestras independientes) μ₁ - μ₂ Muestras aleatorias independientes t o Z
Dos medias (muestras pareadas) μ_d (diferencias) Observaciones pareadas t
Dos proporciones p₁ - p₂ n grande, éxitos ≥ 5 Z
Dos varianzas σ₁²/σ₂² Poblaciones normales F

🔢 Comparación de Dos Medias (Muestras Independientes)

Hipótesis:

Bilateral: \(H_0: \mu_1 = \mu_2\) vs \(H_1: \mu_1 \neq \mu_2\)

Unilateral derecha: \(H_0: \mu_1 \leq \mu_2\) vs \(H_1: \mu_1 > \mu_2\)

Unilateral izquierda: \(H_0: \mu_1 \geq \mu_2\) vs \(H_1: \mu_1 < \mu_2\)

Estadístico de Prueba:

$$t = \frac{(\bar{x}_1 - \bar{x}_2) - (\mu_1 - \mu_2)}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$$

Bajo \(H_0\): \(\mu_1 - \mu_2 = 0\), entonces:

$$t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$$

Grados de libertad (Welch):

$$df = \frac{\left(\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}\right)^2}{\frac{(s_1^2/n_1)^2}{n_1-1} + \frac{(s_2^2/n_2)^2}{n_2-1}}$$

👫 Prueba de Diferencias Pareadas

¿Cuándo usar? Cuando las observaciones están naturalmente pareadas:

  • Mediciones antes/después en los mismos sujetos
  • Pares de gemelos, hermanos
  • Mediciones en pares de objetos similares

Procedimiento:

1. Calcular las diferencias: \(d_i = x_{1i} - x_{2i}\)

2. Calcular \(\bar{d}\) y \(s_d\)

3. Usar prueba t para una muestra sobre las diferencias:

$$t = \frac{\bar{d} - \mu_0}{s_d / \sqrt{n}}$$

donde típicamente \(\mu_0 = 0\) (no hay diferencia)

Grados de libertad: \(df = n - 1\)

📝 Ejemplo Resuelto: Comparación de Dos Grupos

Problema: Un investigador quiere comparar la efectividad de dos métodos de enseñanza. Grupo A (método tradicional): \(\bar{x}_1 = 75\), \(s_1 = 10\), \(n_1 = 30\). Grupo B (método nuevo): \(\bar{x}_2 = 82\), \(s_2 = 12\), \(n_2 = 30\). ¿Hay diferencia significativa? Use α = 0.05.

Paso 1: Hipótesis

$$H_0: \mu_1 = \mu_2 \text{ (no hay diferencia)}$$ $$H_1: \mu_1 \neq \mu_2 \text{ (hay diferencia)}$$

Paso 2: Estadístico

$$t = \frac{75 - 82}{\sqrt{\frac{10^2}{30} + \frac{12^2}{30}}} = \frac{-7}{\sqrt{3.333 + 4.8}} = \frac{-7}{2.85} = -2.46$$

Paso 3: Decisión

Para α = 0.05 bilateral y df ≈ 56, valor crítico ≈ ±2.00

Como |-2.46| > 2.00, rechazamos H₀

p-valor ≈ 0.017

Conclusión

Existe evidencia estadística significativa de que el método nuevo (B) produce mejores resultados que el método tradicional (A). La diferencia promedio de 7 puntos es estadísticamente significativa.

Ejemplo 1: Prueba Z para la Media (Bilateral)

Problema: Una máquina embotelladora llena botellas de refresco que deben contener 500 ml. Se sabe que la desviación estándar del proceso es σ = 10 ml. Un inspector toma una muestra de 36 botellas y encuentra una media de 497 ml. ¿Hay evidencia suficiente para afirmar que la máquina no está funcionando correctamente? Use α = 0.05.

Paso 1: Plantear las Hipótesis

$$H_0: \mu = 500 \text{ ml (la máquina funciona correctamente)}$$ $$H_1: \mu \neq 500 \text{ ml (la máquina no funciona correctamente)}$$

Es una prueba bilateral porque queremos detectar cualquier desviación de 500 ml.

Paso 2: Nivel de Significancia

$$\alpha = 0.05$$

Paso 3: Estadístico de Prueba

Como σ es conocida, usamos la prueba Z:

$$Z = \frac{\bar{x} - \mu_0}{\sigma / \sqrt{n}}$$

Paso 4: Calcular el Estadístico

Datos:

  • \(\bar{x} = 497\) ml
  • \(\mu_0 = 500\) ml
  • \(\sigma = 10\) ml
  • \(n = 36\)
$$Z = \frac{497 - 500}{10 / \sqrt{36}} = \frac{-3}{10/6} = \frac{-3}{1.667} = -1.80$$

Paso 5: Valor Crítico y Decisión

Para una prueba bilateral con α = 0.05:

$$Z_{\alpha/2} = Z_{0.025} = \pm 1.96$$

Región de rechazo: \(Z < -1.96\) o \(Z > 1.96\)

Estadístico calculado: \(Z = -1.80\)

Como \(-1.96 < -1.80 < 1.96\), el estadístico NO cae en la región crítica.

Paso 6: Calcular el p-valor

Para Z = -1.80 en una prueba bilateral:

$$p\text{-valor} = 2 \times P(Z < -1.80) = 2 \times 0.0359 = 0.0718$$

Como \(p = 0.0718 > \alpha = 0.05\), no rechazamos \(H_0\).

Conclusión

Con un nivel de significancia del 5%, NO hay evidencia suficiente para afirmar que la máquina no está funcionando correctamente. La diferencia observada de 3 ml puede atribuirse a la variabilidad aleatoria del muestreo.

Ejemplo 2: Prueba t para la Media (Unilateral)

Problema: Un fabricante afirma que la vida útil promedio de sus baterías es de al menos 40 horas. Se prueba una muestra de 25 baterías y se obtiene una media de 38.5 horas con desviación estándar de 4.2 horas. ¿Contradicen estos datos la afirmación del fabricante? Use α = 0.05.

Paso 1: Plantear las Hipótesis

$$H_0: \mu \geq 40 \text{ horas (afirmación del fabricante)}$$ $$H_1: \mu < 40 \text{ horas (contradicción)}$$

Es una prueba unilateral izquierda porque queremos probar si la media es menor.

Paso 2: Nivel de Significancia

$$\alpha = 0.05$$

Paso 3: Estadístico de Prueba

Como σ es desconocida y n es pequeña, usamos la prueba t:

$$t = \frac{\bar{x} - \mu_0}{s / \sqrt{n}}$$

con \(n - 1 = 24\) grados de libertad.

Paso 4: Calcular el Estadístico

Datos:

  • \(\bar{x} = 38.5\) horas
  • \(\mu_0 = 40\) horas
  • \(s = 4.2\) horas
  • \(n = 25\)
$$t = \frac{38.5 - 40}{4.2 / \sqrt{25}} = \frac{-1.5}{4.2/5} = \frac{-1.5}{0.84} = -1.786$$

Paso 5: Valor Crítico y Decisión

Para una prueba unilateral izquierda con α = 0.05 y 24 gl:

$$t_{0.05, 24} = -1.711$$

Región de rechazo: \(t < -1.711\)

Estadístico calculado: \(t = -1.786\)

Como \(-1.786 < -1.711\), el estadístico SÍ cae en la región crítica.

Paso 6: p-valor

Para t = -1.786 con 24 gl en una prueba unilateral:

$$p\text{-valor} \approx 0.043$$

Como \(p = 0.043 < \alpha = 0.05\), rechazamos \(H_0\).

Conclusión

Con un nivel de significancia del 5%, SÍ hay evidencia suficiente para afirmar que la vida útil promedio de las baterías es menor a 40 horas. Los datos contradicen la afirmación del fabricante.

Ejemplo 3: Prueba Z para la Proporción

Problema: Un candidato político afirma que tiene el apoyo del 60% de los votantes. En una encuesta a 200 votantes, 108 declararon que votarían por él. ¿Hay evidencia suficiente para afirmar que su proporción de apoyo es diferente al 60%? Use α = 0.01.

Paso 1: Plantear las Hipótesis

$$H_0: p = 0.60$$ $$H_1: p \neq 0.60$$

Paso 2: Nivel de Significancia

$$\alpha = 0.01$$

Paso 3: Estadístico de Prueba

$$Z = \frac{\hat{p} - p_0}{\sqrt{\frac{p_0(1-p_0)}{n}}}$$

Paso 4: Calcular el Estadístico

Datos:

  • \(x = 108\) (votantes a favor)
  • \(n = 200\)
  • \(\hat{p} = \frac{108}{200} = 0.54\)
  • \(p_0 = 0.60\)
$$Z = \frac{0.54 - 0.60}{\sqrt{\frac{0.60(0.40)}{200}}} = \frac{-0.06}{\sqrt{\frac{0.24}{200}}} = \frac{-0.06}{\sqrt{0.0012}} = \frac{-0.06}{0.0346} = -1.73$$

Paso 5: Valor Crítico y Decisión

Para una prueba bilateral con α = 0.01:

$$Z_{\alpha/2} = Z_{0.005} = \pm 2.576$$

Región de rechazo: \(Z < -2.576\) o \(Z > 2.576\)

Estadístico calculado: \(Z = -1.73\)

Como \(-2.576 < -1.73 < 2.576\), NO rechazamos \(H_0\).

Paso 6: p-valor

Para Z = -1.73 en una prueba bilateral:

$$p\text{-valor} = 2 \times P(Z < -1.73) = 2 \times 0.0418 = 0.0836$$

Como \(p = 0.0836 > \alpha = 0.01\), no rechazamos \(H_0\).

Conclusión

Con un nivel de significancia del 1%, NO hay evidencia suficiente para afirmar que la proporción de apoyo es diferente al 60%. Aunque la muestra mostró 54%, esta diferencia podría deberse al azar.
